# Data Modeler

## Assignment
Design a MongoDB schema for a blogging platform.

## Models Included
- User
- Category
- Post
- Comment

## Run
1. Open terminal in this folder
2. Run `npm install`
3. Run `npm start`

This runs schema preview even without MongoDB.

## Is MongoDB Installation Compulsory?
No, not compulsory for schema design.

You only need MongoDB when you want to connect and do real database operations.
You can use either:
- Local MongoDB installation, or
- MongoDB Atlas cloud

If you want connection test, set environment variable:
- MONGO_URI=<your_connection_string>
