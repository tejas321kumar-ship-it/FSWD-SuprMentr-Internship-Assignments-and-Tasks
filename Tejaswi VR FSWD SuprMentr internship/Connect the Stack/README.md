# Connect the Stack

Connect React frontend to backend APIs.

## Folder Structure
- frontend (React + Vite)
- backend (Express API)

## Run Backend
1. Open terminal in backend folder
2. Run npm install
3. Run npm start

Backend URL: http://localhost:3020

## Run Frontend
1. Open terminal in frontend folder
2. Run npm install
3. Run npm run dev

Frontend URL: http://localhost:5173

## API Endpoints
- GET /api/health
- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/:id
- DELETE /api/tasks/:id
