# CRUD Lab

Assignment: Implement CRUD operations using Mongoose.

## Files
- app.js
- models/taskModel.js
- .env.example

## Setup
1. Open terminal in this folder.
2. Run npm install
3. Create .env file and add MONGO_URI
4. Run npm start

## Example .env
MONGO_URI=mongodb://127.0.0.1:27017/crud_lab

## Do You Need MongoDB?
Yes, to actually run CRUD with Mongoose, a MongoDB connection is required.

You can use either:
- Local MongoDB installation, or
- MongoDB Atlas connection string

Without MongoDB URI, the script will only show setup guidance and stop.
